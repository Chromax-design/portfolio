1. After you're done with the project, you can Ignore typescript build errors by going into the {next.config.mjs} file and adding the following:

    ####
        typescript: {
            ignoreBuildErrors: true
        }
    ####

2. Also remove the sentry example example file and also its api file, as they are not needed

    - Sentry 
    - Tailwind CSS
